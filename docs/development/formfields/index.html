表单数据说明

mindpage内置了足够多的表单控件，通过数据驱动，可快速配置出组件对应的设置面板。

类型为数组Array，每个item是一个设置项，可嵌套。

例如：

[
    {
        ..."单个控件数据"...
    },
    {
        ..."单个控件数据"...
    },
    {
        ..."单个控件数据"...
    },
]

单个控件格式说明

下面是单个控件的完成格式，以及各个参数的含义说明

数据格式

{
    "label": "背景图片",
    "foreign-key": "card_type",
    "form-type": "input",
    "data-type": "string",
    "require": true,
    "ogName": "newflag",
    "input": {
        "type": "hidden",
        "name": "card_type",
        "value": "PcVideo",
        "placeholder":"",
        "readonly": true,
        "verify":"acttype=number&errmsg=错误提示语"
        "checkType":"weibo",
        "action_type":"changeState",
        "action_data": "target=file&state=1",
        "pic_require":true,
        "file_require":true
    }
}

参数含义

  • label：控件标题，表示控件的含义和作用
  • foreign-key：控件的key值
  • form-type：控件类型，分别为input, image, icon, file, array, object, buttons
  • data-type：表示控件值的类型，分别为string、int、array、object、buttons、在保存数据时，会根据data-type对数据做格式处理
  • require：适用于form_type:input/image的控件用，表示是否为必填
  • ogName：适用于form_type:input且type是radio, checkbox的控件用，值就是对应的key
  • input：
    • type: input控件的类型，分别为hidden, text, number, radio, checkbox, textarea
    • name: 控件的key值，和foreign-key一样
    • value: 控件的值
    • placeholder: 控件输入框内的提示语，建议尽量配置，增加用户可读性。
    • verify: 检验规则，具体参考
    • readonly: 控件是否为只读控件
    • checkType: 通过对应值(mid、uid)，获取平台数据，目前有："weibo"、"user"
    • action_type:
    • action_data:
    • pic_require: 适用于form_type:image的控件，表示是否为必填
    • file_require: 适用于form_type:file的控件，表示是否为必填

控件数据说明

输入框[input]

示例

card中数据：

    "width": 200

表单控件数据：

{
    "label": "图片宽度",
    "form-type": "input",
    "data-type": "int",
    "foreign-key": "width",
    "input": {
        "type": "text",
        "name": "width",
        "verify": "acttype=notEmpty|number&errmsg=请输入图片宽度||图片宽度只能为数字"
    }
}

图片[image]

示例

card中数据：

    "profile_image_url": ""

表单控件数据：

{
    "label": "头像图片",
    "form-type": "image",
    "data-type": "string",
    "foreign-key": "profile_image_url",
    "input": {
        "type": "hidden",
        "name": "profile_image_url"
    }
}

对象[object]

示例

card中数据：

    "btn_url": {
        "scheme": "sinaweibo://sendweibo",
        "url": "http://weibo.cn/qr/sendweibo"
    }

表单控件数据：

{
    "label": "按钮链接",
    "form-type": "object",
    "data-type": "object",
    "foreign-key": "btn_url",
    "params": [
        {
            "label": "端内链接",
            "form-type": "input",
            "data-type": "string",
            "require": true,
            "foreign-key": "scheme",
            "input": {
                "type": "text",
                "name": "scheme",
                "verify": "acttype=scheme&errmsg=请填写正确的scheme地址"
            }
        },
        {
            "label": "端外链接",
            "form-type": "input",
            "data-type": "string",
            "foreign-key": "url",
            "input": {
                "type": "text",
                "name": "url",
                "verify": "acttype=url&errmsg=请填写正确的url"
            }
        }
    ]
}

数组[array]

示例

card中数据：

    "pics": [
        {
            "pic": "http://ww3.sinaimg.cn/large/6455dcafjw1ev822wjdfej20gm06u402.jpg",
            "desc": "图片1",
            "url": "https://www.sina.cn"
        },
        {
            "pic": "http://ww1.sinaimg.cn/large/571e05d1jw1evemtiky4wj20cx08cdgd.jpg",
            "desc": "图片2",
            "url": "https://www.sina.cn"
        }
    ],

表单控件数据：

{
    "label": "图片",
    "form-type": "array",
    "data-type": "array",
    "data-key": "pics",
    "foreign-key": "pics",
    "require": true,
    "array-item": [
        {
            "label": "图片链接",
            "form-type": "image",
            "data-type": "string",
            "require": true,
            "foreign-key": "pic",
            "input": {
                "type": "hidden",
                "name": "pic"
            }
        }, {
            "label": "描述",
            "form-type": "input",
            "data-type": "string",
            "foreign-key": "desc",
            "input": {
                "type": "text",
                "name": "desc",
                "verify": "acttype=max&errmsg=请填写正确的图片信息",
                "vmax": "40"
            }
        }, {
            "label": "图片跳转链接",
            "form-type": "input",
            "data-type": "string",
            "foreign-key": "url",
            "input": {
                "type": "text",
                "name": "url",
                "verify": "acttype=scheme&errmsg=请填写正确的链接地址"
            }
        }
    ]
}

按钮[buttons]

示例

card中数据：

    "buttons": []

表单控件数据：

{
    "label": "按钮",
    "form-type": "buttons",

    "foreign-key": "buttons"
}
