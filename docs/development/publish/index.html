说明

  • 为了便于共享Card组件，使用npm私仓管理MindPage组件。
  • 必须遵守Card开发规范
  • 组件发布需要npm私仓权限，仓库scope为@repoch，可联系张宇(zhangyu19@)帮忙发布，也可联系刘巍峰(weifeng3@)开通私仓@repoch权限。
