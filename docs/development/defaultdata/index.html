组件数据

格式说明

包含了三部分内容，组件类型card_type、默认数据card_data、表单数据form_fields；

{
    card_type: "card00001",
    card_data:{},
    form_fields:[]
}

组件类型

组件的唯一名称，不能和已有Card组件冲突，命名规则如下：

  • 以纯数字命名，格式为cardxxxxx，前2位表示业务，后3位为Card序号;注意：此处的card_type必须和index文件的一致
  • 不同业务线以不同的数字起始
    • 00 - 官方组件
    • 01 - 会员业务
    • 02 - 电商业务
    • 03 - xx业务
    • 10 - 通用card如：动态接口、自定义Card

默认数据

组件渲染所需数据，来源为设置面板配置出来的数据，类型为Object。可自由定制，只要保证数据合法性即可。

示例

{
    pid:"a14df729ly1fe1eb37ypbj20it072dlt",
    href: "http://www.baidu.com"
}

表单数据

mindpage内置了足够多的表单控件，通过数据驱动，可快速配置出组件对应的设置面板。

类型为数组Array，每个item是一个设置项，可嵌套。具体使用说明，请参考表单数据说明

示例

[
    {
        "form-type": "input",
        "data-type": "string",
        "foreign-key": "card_type",
        "input": {
            "type": "hidden",
            "name": "card_type",
            "value": "05"
        }
    },
    {
        "label": "图片",
        "form-type": "image",
        "data-type": "string",
        "require": true,
        "foreign-key": "pic",
        "input": {
            "type": "hidden",
            "name": "pic",
            "verify": "acttype=notEmpty&errmsg=请上传图片"
        }
    },
    {
        "label": "scheme",
        "form-type": "input",
        "data-type": "string",
        "foreign-key": "scheme",
        "input": {
            "type": "text",
            "name": "scheme"
        }
    }
]

整体示例

    card_type: "card00005",
    card_data:{
        pid:"a14df729ly1fe1eb37ypbj20it072dlt",
        href: "http://www.baidu.com"

    form_fields:[
        {
            "form-type": "input",
            "data-type": "string",
            "foreign-key": "card_type",
            "input": {
                "type": "hidden",
                "name": "card_type",
                "value": "05"
            }
        },
        {
            "label": "图片",
            "form-type": "image",
            "data-type": "string",
            "require": true,
            "foreign-key": "pic",
            "input": {
                "type": "hidden",
                "name": "pic",
                "verify": "acttype=notEmpty&errmsg=请上传图片"
            }
        },
        {
            "label": "scheme",
            "form-type": "input",
            "data-type": "string",
            "foreign-key": "scheme",
            "input": {
                "type": "text",
                "name": "scheme"
            }
        }
    ]
};
