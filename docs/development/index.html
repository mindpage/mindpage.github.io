入口文件

组件的入口文件，对外暴露了组件的关键模块

示例：

import Template from './Template/Template';
import icon from './assets/images/icon-card05.png';
import preview from './assets/images/preview-card05.png';
import Preview from './Preview/Preview';
import defaultData from './defaultData';
import formFields from './formfields';
import Form from './Form/Form';

const card = {
    card_type: 'card00005',
    name: '单图',
    desc: '单张图片Card',
    sidebar: {
        icon,
        preview
    },
    preview:Preview,
    template: Template,
    defaultData,
    formFields,
    form:Form
};

export default card;
  • card_typeCard类型。格式为card+id；
  • name Card名称，展示在左侧组件库里。
  • desc.js Card描述，展示在左侧组件库里。
  • sidebar 包括icon和preview，Card在左侧组件库里的展示icon和预览图。
  • preview 自定义的Card展示图标和预览效果。和sidebar作用相同，优先使用sidebar面板。
  • template 组件本身，用于编辑器的预览和线上页面渲染，注意涉及线上页面加载性能，不能包含不必要的内容。
  • defaultData 组件数据，包含了默认数据和右侧编辑面板表单数据。
  • formFields 编辑面板数据，可快速实现编辑面板。
  • form 自定义的编辑面板，如果没有formFields数据，则会取自定义的编辑面板。
